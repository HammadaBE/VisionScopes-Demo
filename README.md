# Visionscopes

## Run locally with Docker

### Requirements
- Docker
- Docker Compose

### Setup

**Before you start:** Make sure Docker Engine is running

### Linux
- git clone https://github.com/hammadabe/visionscopes
- cd visionscopes
- cp .env.example .env
- docker compose up

### Windows
- git clone https://github.com/HammadaBE/VisionScopes-Demo
- cd VisionScopes-Demo
- copy .env.example .env
- docker compose up

### Access
- Frontend: http://localhost:8080
- Backend API: http://localhost:8020/api/
